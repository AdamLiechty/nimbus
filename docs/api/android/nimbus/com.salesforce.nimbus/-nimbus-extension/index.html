NimbusExtension

interface NimbusExtension

Defines an extension which will be bound to the NimbusBridge. Implementations of this interface must also be annotated with Extension.

Functions

cleanup

 open fun cleanup(webView: WebView): Unit

Do any cleanup of the WebView necessary for this extension. Do not remove any javascript interfaces from the WebView here. They will be removed by the NimbusBridge.

customize

 open fun customize(webView: WebView): Unit

Customize the WebView prior to the extension being initialized. Do not add any javascript interfaces to the WebView here. They will be added by the NimbusBridge.
